README
======

This package includes the following files:

|-- Scheduler.c [This file shows three example algorithms for CPU scheduling: First Come First Serve, Shortest Job First (Preemptive), and Priority (Preemptive)]
|-- Makefile [This compiles your Scheduler.c program to a runnable "Scheduler" file]
|-- README.txt [This file]

To compile:
    	make
	OR
	make all

To run:
	./Scheduler <Example file>

For example:
	./Scheduler FirstInput.txt

To clean the folder:
	make clean

To make the appropriate .zip file:
	make zip

To make the appropriate .tar file:
	make tar
